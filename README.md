Dự án Landing Page giới thiệu cuốn sách "The Academic Architect: Kiến Tạo Tương Lai".

## TÍNH NĂNG NỔI BẬT 🚀
- **Hiệu ứng mượt mà**: Sử dụngreveal window on scroll và glassmorphism.
- **Tối ưu tốc độ**: Tải trực tiếp CSS từ CDN thông minh.
- **Tiên tiến**: Thiết kế chuẩn thiết bị Stitch, tối ưu cho trải nghiệm người dùng hiện đại.

## TRIỂN KHAI 🌐
Để triển khai lên Vercel chỉ với 1 cú click, hãy nhấn vào nút **"Deploy with Vercel"** ở trên. Vercel sẽ tự động kết nối với repo GitHub này và cập nhật website mỗi khi bạn có thay đổi.
